DeviceDeck

DeviceDeck is the controller class of device-deck.

init

Initialize DeviceDeck. It means:

  1. initialize BlutoothManager and start discovery immediately
  2. initialize KeyNavigationAdapter
  3. Find all navigable elements and initialize SpatialNavigator

And when it receives device-found event from BluetoothManager, it will create SmartButton for the device.

onBluetoothDeviceFound(device)

Create SmartButton for each device found and append it to div#connected-devices-list or div#newly-found-devices-list

Parameters

  • BluetoothDevice device :

    see BluetoothDevice

refreshDevices

refresh list of devices in div#connected-devices-list and div#newly-found-devices-list
