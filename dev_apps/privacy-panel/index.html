Remote Privacy Protection

  • Please set a pass phrase to secure your remote features (maximum 30 characters).

  • Please enter your pass phrase.

  • You can locate this device via SMS as following: “FmD: LocateDevice YOURPASSPHRASE”

  • You can ring this device via SMS as following: “FmD: SoundAlarm YOURPASSPHRASE”

  • You can lock this device via SMS as following: “FmD: LockDevice YOURPASSPHRASE”

  • You can wipe this device via SMS as following: “FmD: WipeDevice YOURPASSPHRASE”
